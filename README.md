# 🚀 Modern Full-Stack Portfolio

Welcome to my professional portfolio! This is a high-performance, visually stunning web application built with **React**, **Tailwind CSS**, and **Framer Motion**. It features interactive glassmorphism components, a cinematic boot screen, and a dynamic mouse-tracking background.

## ✨ Features

- 🏎️ **Cinematic Boot Screen**: A premium terminal-style preloader with a progress bar and logo reveal.
- 🖱️ **Interactive Background**: A custom Canvas-based background with magnetic particle systems and an aurora spotlight that follows mouse movements.
- 📱 **Fully Responsive**: Optimized for all devices, from mobile to ultra-wide monitors.
- 🎨 **Modern Aesthetics**: Sleek dark mode design using glassmorphism and smooth gradients.
- 🏗️ **Full-Stack Architecture**: Connects to a custom Express/Node.js API with a dedicated Admin Panel for data management.
- 📧 **Direct Contact**: Integrated with **EmailJS** for seamless messaging.

## 🛠️ Technology Stack

- **Frontend**: React 18, Tailwind CSS, Framer Motion
- **Icons**: Lucide React, MD Icons
- **Deployment**: GitHub Pages
- **API Integration**: Axios

## 🚦 Getting Started

### Prerequisites

- **Node.js**: v16.x or higher
- **npm**: v8.x or higher

### Installation

1. **Clone the repository**:
   ```bash
   git clone https://github.com/abhi051002/portfolio.git
   cd portfolio
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Configure Environment Variables**:
   Create a `.env` file in the root directory and add your keys:
   ```env
   REACT_APP_EMAILJS_SERVICE_ID=your_service_id
   REACT_APP_EMAILJS_TEMPLATE_ID=your_template_id
   REACT_APP_EMAILJS_PUBLIC_KEY=your_public_key
   ```

4. **Run the development server**:
   ```bash
   npm start
   ```
   Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

## 📦 Build & Deployment

### Build for Production
To create an optimized production build:
```bash
npm run build
```

### Deploy to GitHub Pages
```bash
npm run deploy
```
